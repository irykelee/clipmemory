| **신호등 버튼** | 타이틀바 내 | 타이틀바 숨김, 통합 툴바 영역 (macOS 26 Liquid Glass) |

| **설정 페이지** | 기본 Form | 사이드바 독립 설정 페이지, 그룹화·간격 최적화 |

## 새로운 기능

### Quick Bar (메뉴바 팝업)
메뉴바 클릭 → NSPopover로 최근 8개 항목 표시 → 클릭 복사 / 검색 / 전체 창 열기

### 길게 누르기 동작 (0.4초)
| 콘텐츠 | 기본 표시 | 길게 누른 후 |
|--------|----------|------------|
| 일반 텍스트 | 처음 200자, 3줄 제한 | 전체 표시 (줄 제한 없음) |
| 민감 콘텐츠 | 마스크 `ab••••••yz` | 원문 표시 + 검색 하이라이트 |
| 이미지 | 썸네일 80px | 300px 확대 |

### 시간 그룹화
클립보드 목록을 생성 시간별로 자동 그룹화: 오늘 / 어제 / 이번 주 / 이번 달 / 이전 (접을 수 있음).

### 글꼴 크기 조정
설정 → 글꼴 크기 → 작게/보통/크게, 모든 UI 텍스트에 적용.

### 단축키 사용자 지정
설정에서 새로운 글로벌 핫키 녹음 (기본값 `Cmd+Ctrl+V`).

### 테마 시스템
설정에서 창 효과 (단색 / 서리 / 울트라)와 외형 (라이트 / 다크 / 시스템 연동) 조정 가능.

## 기능

- 📋 클립보드 기록 (텍스트 / 이미지 / 링크)
- ⭐ 중요 항목 고정, 자동 삭제 방지
- 💾 이미지 암호화 파일 저장, 10MB 제한 돌파
- 🔍 실시간 검색, 전체 언어 정확 하이라이트 (중한일等多바이트 문자対応)
- ✅ 복사 시 녹색 플래시 피드백
- ☑️ 다중 선택 일괄 고정 / 삭제
- 🔒 민감 정보 자동 감지 (25+ 규칙) + AES-256 암호화 + HMAC 인증
- 🔐 비밀번호 관리자가 포그라운드에 있으면 자동 일시 중지, 커스텀 제외 앱 지원
- ⚡ 스마트 중복 제거, 같은 내용은 타임스탬프만 업데이트하여 중복 기록 안 함
- 🔄 복사 루프 방지, App 내에서 복사 시 자동으로 캡처 건너뛰기
- 🔒 보안 우선: 암호화 실패 시 콘텐츠 저장 안 함, 평문 저장 거부
- 🧹 고아 파일 정리, App 실행 시 참조되지 않는 이미지 자동 삭제
- ⚙️ 첫 실행 시 핫키 충돌 자동 감지
- ⌨️ 글로벌 핫키 `Cmd+Ctrl+V`
- 🌍 7개 언어 (简体中文 / 繁體中文 / English / 日本語 / 한국어 / Español / Português)
- 📎 설정 → 정보 → 피드백 보내기 → GitHub Issues

## 사용 방법

| 동작 | 방법 |
|------|------|
| Quick Bar 열기 | 메뉴바 📋 아이콘 / `Cmd+Ctrl+V` |
| Quick Bar에서 복사 | 항목 클릭 / ↑↓ + Enter |
| 전체 창 열기 | Quick Bar → "클립보드 열기" |
| 검색 | 키워드 입력 시 실시간 필터링, 일치 항목 하이라이트 |
| 고정 / 해제 | ⭐ 클릭, 행 더블클릭, 또는 우클릭 메뉴 |
| 삭제 | 🗑 클릭 또는 우클릭 메뉴 |
| 민감 내용 보기 | 0.4초 길게 누르면 표시, 놓으면 숨김 |
| 이미지 확대 | 0.4초 길게 누르면 확대, 놓으면 축소 |
| 전체 텍스트 보기 | 텍스트 항목을 0.4초 길게 누름 |
| 다중 선택 | 체크박스 클릭 |
| 일괄 작업 | 다중 선택 → 일괄 고정 / 삭제 |
| 창 닫기 | `Esc` |
| 기록 지우기 | 상단 도구 모음 🗑 (고정 항목 유지) |

> 💡 고정된 항목은 자동 삭제되지 않습니다. 동일한 내용을 다시 복사하면 중복 없이 타임스탬프만 업데이트됩니다.

## 보안

- **AES-256 + HMAC-SHA256 인증 암호화** — 모든 텍스트와 이미지를 디스크 저장 전 자동 암호화
- **스마트 감지** — 25+ 규칙 (키워드 + 정규식)으로 비밀번호, API 키, 토큰, 개인키, ID 번호 등 자동 식별
- **자동 삭제** — 민감 콘텐츠를 1시간 / 24시간 / 48시간 / 7일 후 자동 삭제 또는 삭제 안 함

## 설정

- 최대 기록 개수 (50 / 100 / 200 / 500)
- 민감 정보 자동 삭제 정책 (1시간 / 24시간 / 48시간 / 7일 / 안 함)
- 언어 전환 (7개 언어)
- 글로벌 핫키 녹음
- 글꼴 크기 (작게 / 보통 / 크게)
- 창 효과 (단색 / 서리 / 울트라)
- 외형 (라이트 / 다크 / 시스템 연동)
- 제외 앱 (클립보드 모니터링에서 제외할 커스텀 앱)

## 설치

```bash
brew tap irykelee/clipmemory https://github.com/irykelee/clipmemory && brew install --cask clipmemory
```

설치 후 App은 `/Applications/ClipMemory.app`에 위치합니다. 실행 후 **화면 오른쪽 상단 메뉴바**의 📋 아이콘을 클릭하여 사용하세요.

또는 [GitHub Releases](https://github.com/irykelee/clipmemory/releases)에서 `.tar.gz`를 다운로드하여 `/Applications/`에 수동으로 압축 해제.
