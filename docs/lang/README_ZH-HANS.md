## 相比于 v1 的改进

| 维度 | v1 | v2 |
|------|----|----|
| **交互入口** | 菜单栏点击 → 菜单列表 → 打开窗口（三步） | 菜单栏点击 → Quick Bar 弹窗（一步） |
| **主界面** | 固定宽度，无侧边栏 | **固定侧边栏**：全部 / 文本 / 图片 / 链接 / 已收藏 / 偏好设置 |
| **类型过滤** | 水平按钮组 | 侧边栏垂直导航列表，带自动计数的条目数量 |
| **时间分组** | 无分组 | 今天 / 昨天 / 更早，支持折叠 |
| **全局快捷键** | 仅 Cmd+Ctrl+V | 支持自定义快捷键（设置页录制） |
| **Quick Bar 弹窗** | 无 | 菜单栏点击弹出最近 8 条内容，搜索、复制、打开窗口一步到位 |
| **搜索高亮** | 在文本上高亮 | 不区分大小写，不乱码 |
| **长按预览** | 无 | 文本按住 0.4s 显示全文、敏感内容按住揭示原文、图片按住放大至 300px |
| **图标排列** | 复选框 + 类型图标 + 星标 + 内容 | 复选框 + 内容 + 星标 + 删除，更简洁 |
| **窗口样式** | 标准窗口 | 毛玻璃效果，更现代 |
| **窗口按钮** | 标题栏内 | 标题栏隐藏，统一工具栏区域显示（macOS 26 Liquid Glass 风格） |
| **Dock 图标** | 一直隐藏 | 打开窗口时出现 Dock 图标，关闭后消失 |
| **悬停高亮** | 不高亮 | 悬停自动高亮 |
| **字体缩放** | 无 | 偏好设置支持小/中/大三档全局字体缩放 |
| **开机自启** | 无（仅菜单栏操作） | 偏好设置新增「登录时启动」 |
| **设置页面** | 基础的 Form | 侧边栏独立设置页，分组标明、间距优化 |

## 新增功能

### Quick Bar（菜单栏弹窗）

点击菜单栏图标 → NSPopover 弹窗 → 最近 8 条内容 → 点击复制 / 搜索 / 打开完整窗口

### 长按操作（0.4s）

| 内容类型 | 默认显示 | 长按后 |
|---------|---------|--------|
| 普通文本 | 前 200 字符，3 行截断 | 全文显示（无行数限制） |
| 敏感内容 | 遮罩 `ab••••••yz` | 揭示原文 + 搜索高亮 |
| 图片 | 缩略图 80px | 放大至 300px |

### 时间线分组

剪贴板列表自动按创建时间分组：今天 / 昨天 / 更早，可折叠。

### 全局字体缩放

偏好设置 → 字体大小 → 小/中/大，所有界面文字同步缩放。

### 快捷键自定义

偏好设置可录制新的全局快捷键替代默认 `Cmd+Ctrl+V`。

### 主题系统

偏好设置可调整窗口效果（实心 / 毛玻璃 / 高透）和外观（浅色 / 深色 / 跟随系统）。

## 功能特点

- 📋 剪贴板历史记录（文本 / 图片 / 链接）
- ⭐ 收藏重要片段，不被自动清理
- 💾 图片加密文件存储，突破 10MB 限制
- 🔍 实时搜索，所有语言高亮支持（含中日韩等多字节字符）
- ✅ 复制绿色闪烁反馈
- ☑️ 多选批量收藏 / 删除
- 🔒 敏感信息自动检测（25+ 条规则）+ AES-256-GCM（v2）+ 兼容旧版 AES-CBC+HMAC-SHA256
- 🔐 密码管理器在前台时自动暂停捕获，支持自定义排除 App
- ⚡ 智能去重，相同内容只更新时间戳，不重复记录
- 🔄 复制循环拦截，从 App 内复制时自动跳过
- 🔒 安全优先：加密失败时内容不落地，拒绝明文存储
- 🧹 孤立文件清理，App 启动时自动清理无引用的图片
- ⚙️ 首次启动时自动检测快捷键冲突
- ⌨️ 全局快捷键 `Cmd+Ctrl+V`
- 🌍 7 种语言（简体中文 / 繁體中文 / English / 日本語 / 한국어 / Español / Português）
- 📎 偏好设置 → 关于 → 发送反馈 → GitHub Issues

## 使用方法

| 操作 | 方式 |
|------|------|
| 弹出 Quick Bar | 左键点击菜单栏 📋 图标 / `Cmd+Ctrl+V` |
| 复制 Quick Bar 条目 | 点击条目 / 键盘 ↑↓ + Enter |
| 打开完整窗口 | Quick Bar → "打开完整窗口" |
| 搜索 | 输入关键词实时过滤，匹配高亮 |
| 收藏 / 取消收藏 | 点击 ⭐、双击条目、或右键菜单 |
| 删除 | 点击 🗑 或右键菜单 |
| 查看敏感内容 | 按住 0.4s 揭示原文，松开隐藏 |
| 查看图片原图 | 按住 0.4s 放大，松开恢复 |
| 查看全文 | 按住 0.4s 显示全部文字 |
| 多选 | 单击复选框进入多选模式 |
| 批量操作 | 多选后批量收藏 / 删除 |
| 关闭窗口 | `Esc` |
| 清空历史 | 顶栏 🗑 按钮（保留收藏条目） |

> 💡 已收藏的条目不会被自动清理。复制相同内容不会重复记录，只更新时间戳提到顶部。

## 安全特性

- **AES-256-GCM（v2）+ 兼容旧版 AES-CBC+HMAC-SHA256** — 所有文本和图片内容在存入磁盘前自动加密
- **智能检测** — 25+ 条检测规则（关键词 + 正则），自动识别密码、API 密钥、token、私钥、身份证号、银行卡号等
- **自动清理** — 敏感内容可设置 1 小时 / 24 小时 / 48 小时 / 7 天后自动清除，或不自动清除

## 安装

```bash
brew tap irykelee/clipmemory https://github.com/irykelee/clipmemory && brew install --cask clipmemory
```

安装后 App 在 `/Applications/ClipMemory.app`。启动后看**屏幕右上角菜单栏**的 📋 图标，点击即可使用。

或从 [GitHub Releases](https://github.com/irykelee/clipmemory/releases) 下载 `.tar.gz` 手动解压到 `/Applications/`。
