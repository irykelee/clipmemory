| 項目 | v1 | v2 |

| **メイン画面** | 固定幅、サイドバーなし | **固定サイドバー**：すべて / テキスト / 画像 / リンク / ピン留め / 環境設定 |
| **タイプフィルター** | 横並びボタン | サイドバーの垂直リスト（件数表示）|
| **時間グループ** | なし | 今日 / 昨日 / 今週 / 今月 / 以前（折りたたみ可能）|
| **グローバルホットキー** | Cmd+Ctrl+V のみ | カスタマイズ可能（設定で録音）|
| **Quick Bar** | なし | 8件の最近のアイテムをポップアップ表示、検索・コピー・ウィンドウ表示 |
| **検索ハイライト** | テキスト上にハイライト | `offsetByCharacters` で正確な位置、日中英対応 |
| **ロングプレス** | なし | テキスト→全文、機密→表示、画像→拡大（0.4秒長押し）|
| **アイコン配置** | チェックボックス + タイプアイコン + 星 + 内容 | チェックボックス + 内容 + 星 + 削除（タイプアイコン削除）|

| **信号ボタン** | タイトルバー内 | タイトルバー非表示、統一ツールバー領域（macOS 26 Liquid Glass）|
| **Dock アイコン** | 常に非表示 | ウィンドウ表示中に出現、閉じると消える |
| **ホバーハイライト** | なし | ホバーで自動ハイライト |
| **フォントサイズ** | なし | 小/中/大の3段階、全UIに適用 |
| **ログイン時起動** | なし（メニューのみ） | 設定で有効化 |
| **設定画面** | 基本的な Form | サイドバーの独立設定ページ、グループ化・間隔最適化 |

## 新機能

### Quick Bar（メニューバーポップアップ）
メニューバークリック → NSPopover で最近8件表示 → クリックでコピー / 検索 / フルウィンドウ表示

### ロングプレス操作（0.4秒）
| コンテンツ | デフォルト表示 | 長押し後 |
|-----------|--------------|---------|
| 通常テキスト | 最初の200文字、3行で切り捨て | 全文表示（行数制限なし）|
| 機密コンテンツ | マスク `ab••••••yz` | 原文表示 + 検索ハイライト |
| 画像 | サムネイル 80px | 300px に拡大 |

### 時間グループ
クリップボードリストを自動作成時間でグループ化：今日 / 昨日 / 今週 / 今月 / 以前（折りたたみ可能）。

### フォントスケーリング
設定 → フォントサイズ → 小/中/大、全UIテキストに適用。

### ホットキーカスタマイズ
設定で新しいグローバルホットキーを録音（デフォルトは `Cmd+Ctrl+V`）。

### テーマシステム
設定でウィンドウ効果（ソリッド / フロスト / ウルトラ）と外観（ライト / ダーク / システム連動）を調整可能。

## 機能

- 📋 クリップボード履歴（テキスト / 画像 / リンク）
- ⭐ 大事なアイテムをピン留め、自動削除されない
- 💾 画像は暗号化ファイルとして保存、10MB制限を突破
- 🔍 リアルタイム検索、日中英で正確にハイライト
- ✅ コピー時に緑のフラッシュフィードバック
- ☑️ 複数選択で一括ピン留め / 削除
- 🔒 機密情報自動検出（25+ルール）+ AES-256暗号化 + HMAC認証
- ⌨️ グローバルホットキー `Cmd+Ctrl+V`
- 🌍 7言語対応（简体中文 / 繁體中文 / English / 日本語 / 한국어 / Español / Português）
- 📎 設定 → このAppについて → フィードバック → GitHub Issues

## 使い方

| 操作 | 方法 |
|------|------|
| Quick Bar を開く | メニューバー 📋 アイコン / `Cmd+Ctrl+V` |
| Quick Bar からコピー | アイテムをクリック / ↑↓ + Enter |
| フルウィンドウを開く | Quick Bar → "クリップボードを開く" |
| ピン留め / 解除 | ⭐ をクリック、行をダブルクリック、または右クリックメニュー |
| 削除 | 🗑 をクリック、または右クリックメニュー |
| 機密内容を表示 | 0.4秒長押しで表示、離すと非表示 |
| 画像を拡大 | 0.4秒長押しで拡大、離すと縮小 |
| 全文表示 | テキストアイテムを0.4秒長押し |
| 複数選択 | チェックボックスをクリック |
| 一括操作 | 複数選択 → 一括ピン留め / 削除 |
| ウィンドウを閉じる | `Esc` |
| 履歴をクリア | 上部ツールバーの 🗑（ピン留めは保持）|

> 💡 ピン留めしたアイテムは自動削除されません。同じ内容を再コピーすると重複せずタイムスタンプのみ更新されます。

## セキュリティ

- **AES-256 + HMAC-SHA256 認証暗号化** — すべてのテキストと画像をディスク保存前に自動暗号化
- **スマート検出** — 25+ルール（キーワード + 正規表現）でパスワード、APIキー、トークン、秘密鍵、ID番号などを自動識別
- **自動クリア** — 機密コンテンツを1時間 / 24時間 / 48時間 / 7日後に自動削除、または削除しない設定も可能

## 設定

- 最大履歴件数（50 / 100 / 200 / 500）
- 機密情報自動クリア（1時間 / 24時間 / 48時間 / 7日 / なし）
- 言語切替（7言語）
- グローバルホットキー録音
- フォントサイズ（小 / 中 / 大）
- ウィンドウ効果（ソリッド / フロスト / ウルトラ）
- 外観（ライト / ダーク / システム連動）

## インストール

```bash
brew tap irykelee/clipmemory https://github.com/irykelee/clipmemory && brew install --cask clipmemory
```

インストール後、App は `/Applications/ClipMemory.app` に配置されます。起動後、**画面右上のメニューバー**にある 📋 アイコンをクリックして使用開始。

または [GitHub Releases](https://github.com/irykelee/clipmemory/releases) から `.tar.gz` をダウンロードして `/Applications/` に手動配置。
