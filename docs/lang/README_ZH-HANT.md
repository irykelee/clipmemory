## 相比於 v1 的改進

| 項目 | v1 | v2 |
|------|----|----|
| **操作入口** | 選單列點擊 → 選單 → 開視窗（三步） | 選單列點擊 → Quick Bar 彈窗（一步）|
| **主介面** | 固定寬度，無側邊欄 | **固定側邊欄**：全部 / 文字 / 圖片 / 連結 / 已釘選 / 偏好設定 |
| **類型篩選** | 水平按鈕組 | 側邊欄垂直列表，附項目數量 |
| **時間分組** | 無 | 今天 / 昨天 / 更早（可折疊）|
| **全域熱鍵** | 僅 Cmd+Ctrl+V | 支援自訂快捷鍵（設定頁錄製）|
| **Quick Bar 彈窗** | 無 | 選單列點擊彈出最近 8 條，搜尋複製開視窗一步到位 |
| **搜尋高亮** | 在文字上高亮 | 精確匹配，不亂碼 |
| **長按預覽** | 無 | 文字 0.4s 顯示全文、敏感內容揭示原文、圖片放大至 300px |
| **圖示排列** | 核取方塊 + 類型圖示 + 星號 + 內容 | 核取方塊 + 內容 + 星號 + 刪除，更簡潔 |
| **視窗樣式** | 標準視窗 | 毛玻璃效果，更現代 |
| **紅綠燈** | 標題列內 | 標題列隱藏，統一工具欄區域顯示（macOS 26 Liquid Glass 風格）|
| **Dock 圖示** | 一直隱藏 | 開啟視窗時出現 Dock 圖示，關閉後消失 |
| **右鍵高亮** | 不高亮 | 懸停自動高亮 |
| **字體縮放** | 無 | 設定頁支援小/中/大三檔全域字體縮放 |
| **開機自啟** | 無（僅選單列操作） | 設定頁新增「登入時啟動」|
| **設定頁佈局** | 基礎的 Form | 側邊欄獨立設定頁，分組標明、間距最佳化 |

## 新增功能

### Quick Bar（選單列彈窗）
點擊選單列圖示 → NSPopover 彈窗 → 最近 8 條內容 → 點擊複製 / 搜尋 / 開啟完整視窗

### 長按操作（0.4s）
| 內容類型 | 預設顯示 | 長按後 |
|---------|---------|--------|
| 一般文字 | 前 200 字元，3 行截斷 | 全文顯示（無行數限制）|
| 敏感內容 | 遮罩 `ab••••••yz` | 揭示原文 + 搜尋高亮 |
| 圖片 | 縮圖 80px | 放大至 300px |

### 時間線分組
剪貼簿列表自動按建立時間分組：今天 / 昨天 / 更早，可折疊。

### 全域字體縮放
設定頁 → 字體大小 → 小/中/大，所有介面文字同步縮放。

### 快捷鍵自訂
設定頁可錄製新的全域快捷鍵替代預設 `Cmd+Ctrl+V`。

### 主題系統
設定頁可調整視窗效果（實心 / 毛玻璃 / 高透）和外觀（淺色 / 深色 / 跟隨系統）。

## 功能特點

- 📋 剪貼簿歷史記錄（文字 / 圖片 / 連結）
- ⭐ 釘選重要片段，不會被自動清理
- 💾 圖片加密檔案儲存，突破 10MB 限制
- 🔍 即時搜尋，所有語言精確高亮（含中日韓等多位元組字元）
- ✅ 複製綠色閃爍反饋
- ☑️ 多選批次釘選 / 刪除
- 🔒 敏感資訊自動檢測（25+ 條規則）+ AES-256 加密 + HMAC 認證
- 🔐 密碼管理器在前台時自動暫停捕獲，支援自訂排除 App
- ⚡ 智慧去重，相同內容只更新時間戳，不重複記錄
- 🔄 複製循環攔截，從 App 內複製時自動跳過
- 🔒 安全優先：加密失敗時內容不落地，拒絕明文儲存
- 🧹 孤立檔案清理，App 啟動時自動清理無參考的圖片
- ⚙️ 首次啟動時自動檢測快捷鍵衝突
- ⌨️ 全域快捷鍵 `Cmd+Ctrl+V`
- 🌍 7 種語言（简体中文 / 繁體中文 / English / 日本語 / 한국어 / Español / Português）
- 📎 設定 → 關於 → 傳送回饋 → GitHub Issues

## 使用方法

| 操作 | 方式 |
|------|------|
| 彈出 Quick Bar | 左鍵點擊選單列 📋 圖示 / `Cmd+Ctrl+V` |
| 複製 Quick Bar 條目 | 點擊條目 / 鍵盤 ↑↓ + Enter |
| 開啟完整視窗 | Quick Bar → "開啟剪貼簿" |
| 搜尋 | 輸入關鍵詞即時過濾，符合處高亮 |
| 釘選 / 取消釘選 | 點擊 ⭐、雙擊條目、或右鍵選單 |
| 刪除 | 點擊 🗑 或右鍵選單 |
| 檢視敏感內容 | 按住 0.4s 揭示原文，鬆開隱藏 |
| 檢視圖片原圖 | 按住 0.4s 放大，鬆開恢復 |
| 檢視全文 | 按住 0.4s 顯示全部文字 |
| 多選 | 點擊核取方塊進入多選模式 |
| 批次操作 | 多選後批次釘選 / 刪除 |
| 關閉視窗 | `Esc` |
| 清空歷史 | 頂欄 🗑 按鈕（保留已釘選條目）|

> 💡 已釘選的條目不會被自動清理。複製相同內容不會重複記錄，只更新時間戳提到頂部。

## 安全特性

- **AES-256 + HMAC-SHA256 認證加密** — 所有文字和圖片內容在存入磁碟前自動加密
- **智慧檢測** — 25+ 條檢測規則（關鍵詞 + 正規式），自動識別密碼、API key、token、私鑰、身分證號、銀行卡號等
- **自動清理** — 敏感內容可設定 1 小時 / 24 小時 / 48 小時 / 7 天後自動清除，或永不自動清除

---

## 設定選項

- 歷史記錄最大條數（50 / 100 / 200 / 500 條）
- 敏感資訊清除策略（1 小時 / 24 小時 / 48 小時 / 7 天 / 不自動清除）
- 語言切換（7 種語言）
- 全域快捷鍵錄製
- 字體大小（小 / 中 / 大）
- 視窗效果（實心 / 毛玻璃 / 高透）
- 外觀（淺色 / 深色 / 跟隨系統）
- 排除 App（自訂不監控的 App）

## 安裝

```bash
brew tap irykelee/clipmemory https://github.com/irykelee/clipmemory && brew install --cask clipmemory
```

安裝後 App 在 `/Applications/ClipMemory.app`。啟動後看**螢幕右上角選單列**的 📋 圖示，點擊即可使用。

或從 [GitHub Releases](https://github.com/irykelee/clipmemory/releases) 下載 `.tar.gz` 手動解壓到 `/Applications/`。

---

## 開發

```bash
brew install swiftlint xcodegen
xcodegen generate
xcodebuild -scheme ClipMemory -configuration Release
```
