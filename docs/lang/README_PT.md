## Melhorias em relação ao v1

| Aspecto | v1 | v2 |
|---------|----|----|
| **Interação** | Clique menu → menu → janela | Quick Bar pop-up (1 passo) |
| **Janela principal** | Largura fixa, sem barra lateral | **Navegação com barra lateral** |
| **Filtro de tipo** | Botões horizontais | Lista vertical na barra lateral |
| **Agrupamento por tempo** | Nenhum | Hoje / Ontem / Esta semana / Este mês / Anterior |
| **Visualização longa** | Nenhuma | Texto completo, conteúdo sensível, imagem ampliada (0.4s) |
| **Estilo de janela** | NSWindow padrão | Efeito vidro estilo Safari 26 |
| **Tamanho da fonte** | Nenhum | Pequeno / Médio / Grande nas configurações |

## Novos recursos

- **Quick Bar**: Clique no ícone da barra de menu → últimos 8 itens → copiar / pesquisar
- **Pressionar e segurar**: 0.4s para ver texto completo, revelar conteúdo sensível, ampliar imagem
- **Agrupamento por tempo**: Agrupamento automático por data de criação (recolhível)
- **Escala de fonte**: Ajuste de tamanho do texto da interface nas configurações
- **Atalho personalizável**: Gravar novo atalho global nas configurações

## Instalação

```bash
brew tap irykelee/clipmemory https://github.com/irykelee/clipmemory && brew install --cask clipmemory
```

Encontre o ícone 📋 na **barra de menu** (canto superior direito). Ou baixe do [GitHub Releases](https://github.com/irykelee/clipmemory/releases).

## Requisitos

macOS 13.0 (Ventura) ou superior
