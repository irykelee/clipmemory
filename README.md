# ClipMemory 剪忆

本地剪贴板历史管理器 - 永久记忆你的每一次复制

## 功能特点

- 📋 剪贴板历史记录（文本/链接）
- ⭐ 固定重要片段，永不丢失
- 🔍 快速搜索历史记录
- 🔒 敏感信息自动保护
- ⌨️ 全局快捷键 Cmd+Ctrl+V 呼出

## 安装

```bash
brew install --cask https://raw.githubusercontent.com/irykelee/clipmemory/main/clipmemory.rb
```

## 系统要求

- macOS 10.15 (Catalina) 或更高版本
